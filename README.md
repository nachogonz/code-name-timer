# Code-Name Voice Timer

Quick Next.js app for a 2-team chess-clock style Code-Name timer.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy on Vercel

1. Push this folder to GitHub
2. Import the repo in Vercel
3. Deploy with the default settings

## Commands

- start
- finish
- stop
- resume
- reset
- status

## Notes

Voice recognition works best after deployment over HTTPS.
Local `file://` usage can cause microphone/session issues in browsers.
